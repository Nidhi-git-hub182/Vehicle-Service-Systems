# Vehicle Service Management System
Successfully implemented the full-stack requirements.

### How to Run:
1. Backend: `python manage.py runserver` (Runs on port 8000)
2. Frontend: `npm start` (Runs on port 3000)

### Features Captured in Screenshots:
- **Revenue Logic:** Handled via Django models and visualized with Recharts.
- **System Architecture:** Full-stack integration between React and Django REST Framework.
